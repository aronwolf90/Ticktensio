Set elements for start page and subpages in admin panel

As admin of Filmtimer I want to be able to set important text blocks and pictures for start page and subpages at admin panel.

ACs:

  • Only text blocks (only possible formatting: setting </br>-Tags for line break) and images (no restriction of uploading) can be set.
  • There should be a new element at menu in admin panel called "page-elements" where you can set these elements.
  • There you should find a table with following columns:
  • language (DE, EN)
  • page name (start page, freelancers, models, extras, companies, registration)
  • page field (names listed in this issue in quotation marks)
  • element type (text or image)
  • show/ edit buttons (as at pages) => no delete button necessary, because every element has to set for one time
  • In "add and edit mode" you have following fields, where you can change things:
  • Dropdown-menu: language
  • Dropdown-menu: page name
  • Dropdown-menu: page field (shown all field for chosen page)
  • Text field (or if field type is picture picture upload field)

Elements at start page:

  • Following elements of start page should be able to change:
  • img: "Large image at the top of the page"
  • text: "Text on image at the top of the page"
  • text: "Button-text on image at the top of the page"
  • text: "Heading on blue background directly under the first image" (at the moment "Was bietet Filmtimer?")
  • text: "Text on blue background directly under the first image" (at the moment "Filmtimer.com bietet...")
  • text: "Heading for chosing profiles" (at the moment "für wen ist Filmtimer")
  • text: "Introduce extras" (at the moment "gezielt gefunden...")
  • text: "Introduce models" (at the moment "gezielt gefunden...")
  • text: "Introduce freelancers" (at the moment "Freiberufler, sowie...")
  • text: "Introduce companies" (at the moment "Ihr Angebot und...")
  • text: "Heading people-slider (at the moment "Wir sind Filmtimer")
  • text: "Text people-slider (at the moment "Das erste Jahr ist...")
  • text: "Button people-slider (at the moment "Also los! Jetzt registrieren")
  • text: "Heading call to action" (at the moment "Sie wollen auch ein Teil von...")
  • text: "Button call to action" (at the moment "Melden Sie sich gleich an")

Elements at freelancers page:

  • Following elements of start page should be able to change:
  • img: "Large image at the top of the page"
  • text: "Large text on image at the top of the page"
  • text: "Button-text on image at the top of the page"
  • text: "Heading on blue background directly under the first image" (at the moment "Profitieren Sie von den Vorteilen...")
  • text: "Text on blue background directly under the first image" (at the moment "Profitieren Sie noch heute von den...")
  • text: "Button on blue background directly under the first image" (at the moment "Jetzt kostenlos anmelden")
  • text: "Heading more information on white background" (at the moment "Ihr Profil als...")
  • text: "Text more information on white background" (at the moment "Haben Sie bereits...")
  • text: "Heading people-slider (at the moment "Wir sind Filmtimer")
  • text: "Text people-slider (at the moment "Registrieren Sie sich jetzt...")
  • text: "Button people-slider (at the moment "Jetzt kostenlos registrieren")
  • text: "Heading functions (at the moment "Diese Funktionen bietet ihnen...")
  • text: "Text functions (at the moment "Sie künnen zwischen einem...")
  • text: "Heading extended profile (at the moment "Erweitertes Profil")
  • text: "Text extended profile (at the moment "12 Monate für...")
  • text: "Heading basic profile (at the moment "Basisprofil")
  • text: "Text basic profile (at the moment "kostenlos")
  • text: "Heading call to action" (at the moment "Sie wollen auch ein Teil von...")
  • text: "Button call to action" (at the moment "Melden Sie sich gleich an")
